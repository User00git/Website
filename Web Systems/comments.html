Personal Reflection

Designing this website has been a journey involving the blend of my creative skills alongside my technical expertise. I aimed to deliver a website that visually pushed the boundaries of the criteria with a unique outlook and captured the users' attention from the moment they open the page. Throughout the creation of this website, I have developed a greater understanding of the complexities and intricases within web design from creating a functional layout to incorporating small features that improve the user interface.

Balancing aesthetics with functionality provided its challenge as the website was made to have a strict container within the body. I had to ensure the site was able to allow the user to function within the restrictive space and look appealing on a multitude of devices and web-browsers. The project showcases the merging of creativity and technological expertise in the digital realm.

Aesthetics

The user interface is crafted with a focus on the minimalistic design principles one that mimcs a professional high-end website, it favors a clean and uncluttered user experience that prioritises content readability and ease of navigation.

Colour Theory

The colour scheme predominately utilises a white background, with black text. The use of white serves as a neutral backdrop that enhances readability and creates a sense of sleek and simplicity to the user. The red logo and backdrop when hovering over the navigation bar helps keep the user entertained as red is a strong colour and complements white very well. It also helps draw attention to important elements such as titles and tables. The hue of red and white is vibrant to purposely maintain the user's engagement.

To avoid overwhelming the reader, red is only used in important aspects and two lighter hues are added for the design of the tables. Together, the combination of red and white in the color scheme achieves a balance between urgency and clarity.

Font

For the webpage's font, there is a uniform and clear style. The main text uses 'Courier New', which is a monospaced typeface that offers easy-to-read lines. This size of 12px for the text ensures content can be seen without being too big or small on screen. On the other hand, headers are styled in a way to make them very noticeable. The header's h1 element has a size of 28px which makes appear bold and big. Different font types and styles like Arial in boxed-header class and Courier New for h3 further create an interesting visual effect yet still easy to read style.

Interactivity

The header is in the middle, a unique design for the interface, this makes it more noticeable and gives a place for main title to become focus point of page. Also, the footer has navigation links so users can effortlessly visit various parts of the website. These links show hover effects which give visual response when a user's mouse pointer is on them. This adds to the interactive experience and makes it easy for users to navigate through different sections of this webpage or site in general.

Structure

The body uses flexbox layout which helps in aligning content correctly and adjusting it to various screen sizes for a responsive design. The main area for content is put in a 'container'; it shows at center and has padding around to give visual separation from other elements around it. Tables have a design feature with collapsed borders and alternate background colors, making it easier to read. The structured layout makes the webpage clearer and more usable by users because it ensures they can access or move around content easily.

Accessibility

Adherence to W3C's Web Content Accessibility Guidelines (WCAG) ensures that the website is accessible to a wider range of people, including those with disabilities.

Screen Reader

Proper use of HTML semantic elements and ARIA labels ensures the site is navigable and comprehensible when using screen readers.

Font Size

All text is displayed in a readable font size, with the option for users to increase or decrease the size as needed using browser functionality.

Font Spacing

Adequate spacing between lines and characters helps to improve readability and prevent visual crowding.

Font Selection

Fonts have been chosen for their readability on digital screens, including considerations for users with dyslexia or other reading difficulties.
